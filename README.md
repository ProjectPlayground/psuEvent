# psuEvent
Using Ionic 2 + AngularJS 2 with BarcodeScanner and export CSV file

### To Run to see UI
``` 
ionic serve -l
```
### To Build app
To android platform
Add android platform to your application
``` 
ionic platform add android
```
Build the android app
``` 
ionic build android
```
Run the android app
``` 
ionic run android
```
